## ğŸŒ Apa Itu HTML?

**HTML (HyperText Markup Language)** adalah bahasa standar untuk membuat dan menyusun halaman web.

HTML digunakan untuk:
- Struktur konten halaman web
- Menampilkan teks, gambar, link, video, dan elemen lainnya
